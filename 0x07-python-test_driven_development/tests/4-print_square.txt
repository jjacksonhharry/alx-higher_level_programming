Test cases for 4-print_square.py module
====================================================

The 4-print_square.py module function that prints a square with the character #

Usage
=====

Squares are printed using the ``#`` character. ``size``
represents the height and width of the square.

::

    >>> print_square = __import__('4-print_square').print_square
    >>> print_square(3)
    ###
    ###
    ###

::

    >>> print_square(4)
    ####
    ####
    ####
    ####

::

    >>> print_square(7)
    #######
    #######
    #######
    #######
    #######
    #######
    #######

If ``size`` is zero, the function prints nothing.

::

    >>> print_square(0)
